# Cuckoo 版本规划与路线图

**更新时间**: 2026-05-22
**当前版本**: v2.0.0
**开发模式**: 审计驱动开发 — 每版结束后执行 12 维度审计，问题分级纳入下版 backlog

---

## ✅ 已发布版本

### v2.0.0（2026-05-22）— 营销协同

| 功能 | 详情 |
|------|------|
| 顾客积分 / 集点系统 | 顾客档案（姓名/手机）、积分余额、消费累计、集点历史、手动调整 |
| 单品退单 | `order_items.refunded` 标记，逐品退款，累计退款金额写回订单 |
| 角色权限系统 | 老板/收银/厨师/仓库 PIN 保护，侧边栏菜单过滤，路由级访问控制 |
| 局域网多设备同步 | 零依赖 TCP/HTTP 主机服务，从机 4 秒轮询合并订单状态 |

### v1.6.0（2026-05）— 快捷与仪表板

| 功能 | 详情 |
|------|------|
| POS 常用品项快捷区 | 菜单页可标记常用，POS 显示"★ 常用"分类快捷入口 |
| 仪表板强化 | 本周 vs 上周销售对比（%）、今日热销 Top 5、待出餐数、库存预警 |

### v1.5.0（2026-04）— 体验打磨

| 功能 | 详情 |
|------|------|
| 订单筛选 | 日期区间 + 状态多选过滤 |
| 食材到期通知 | 启动时检测 3 天内到期批次，生成系统通知 |
| 取消原因记录 | 取消对话框新增原因输入，写入 `orders.cancel_reason` |
| 退款金额 UI | 取消已付款订单时显示退款金额输入框 |
| 已付款收据打印 | 订单列表「打印」按钮，调用 Feie/LAN 打印机 |
| 时段/星期报表 | 报表页新增"时段分析"标签，24 小时 + 7 天柱状图 |

### v1.4.x（2026-04）— 打印完善 & 数据安全

| 功能 | 详情 |
|------|------|
| 收据打印 | 支持飞鹅 + LAN 两种打印机，可配置模板 |
| 入库标签打印 | 采购收货自动打印 TSPL 批次标签 |
| 工作站绑定打印机 | 每个 KDS 工作站可独立指定打印机 |
| 本地自动备份 | 启动时自动备份 SQLite，支持手动触发 |

### v1.0.x ~ v1.3.x（2026-03 ~ 2026-04）

- P0/P1 生产阻断缺陷修复（41 项）
- React Router 路由化、Hooks 架构、统一类型定义
- BOM 配方编辑、半成品嵌套、成本核算
- KDS 厨房显示、FIFO/FEFO 批次扣减
- 采购单、生产单、库存盘点全链路

## 🚫 明确不做（v2.x 范围）

- 在线支付集成 — 超出本地优先架构
- 多语言 i18n — 当前仅面向中文用户
- WebSocket 实时推送 — 4 秒轮询已满足烘焙店场景

## 审计驱动开发

每次版本开发结束后执行标准化审计流水线（12 个维度），将发现的问题分级纳入下版 backlog：

```
开发完成 → 审计流水线（12 维度）→ 问题分级 → 纳入 backlog → 下版本修复
```

详见 [docs/audit-workflow.md](docs/audit-workflow.md)
