## 快速开始

### 安装

1. 从 [Releases](https://github.com/0xRyanlee/hyphen-cuckoo/releases) 下载对应平台的安装包
2. **macOS**：打开 `.dmg`，将 Cuckoo 拖入「应用程序」文件夹
3. **Windows**：双击 `.exe` 安装程序，按提示完成安装

> ⚠️ macOS 首次打开可能提示"无法验证开发者"。前往 **系统设置 → 隐私与安全性**，点击「仍要打开」。

### 首次使用建议流程

```
1. 录入材料（原料档案）
2. 创建配方（BOM 物料清单）
3. 设置菜单（关联配方、定价）
4. 开始点餐
```

## 日常操作：点餐与出餐

### POS 点餐

1. 进入「POS 点餐」页面
2. 左侧选择菜品分类，点击菜品加入购物车
3. 如有规格（大杯/小杯等），会弹出规格选择
4. 支持「加料/去料」操作（如：加珍珠、去冰）
5. 选择用餐类型：堂食 / 外带 / 外送
6. 确认购物车内容，点击「提交订单」

> 💡 购物车会自动保存到本地，关闭页面后再次打开会恢复。

### KDS 厨房显示

1. 进入「KDS」页面，系统每 15 秒自动刷新
2. 新订单以卡片形式展示，按工作站分组
3. 点击「开始制作」标记工单为进行中
4. 制作完成后点击「完成出餐」
5. 超过 15 分钟未完成的工单会以红色高亮警告

> 💡 当所有工作站的工单都完成时，订单状态自动变为「就绪」，库存同步扣减。

### 订单管理

- 在「订单」页面可查看所有历史订单
- 支持按状态筛选（进行中 / 已完成 / 已取消）
- 点击订单可查看详情，包含菜品、金额、时间

## 菜单管理

### 创建菜品

1. 进入「菜单」页面
2. 点击「新增菜品」
3. 填写：名称、分类、售价
4. （可选）关联配方 — 关联后系统可自动计算成本和扣减库存
5. （可选）添加规格（如：大/中/小杯，各有不同价格）

### 菜品与配方的关系

- 每个菜品可关联一个配方
- 关联后，售出此菜品时系统会自动：
  - 计算食材成本
  - 在厨房完成出餐时扣减对应库存
- 未关联配方的菜品仍可点餐，但不会有库存扣减

## 配方管理

### 核心概念

- **配方** = 一道菜品的物料清单（BOM）
- **配方明细** = 所需材料及用量
- **半成品配方** = 可被其他配方引用的中间产物（如：酱汁、高汤）

### 创建配方

1. 进入「配方」页面
2. 点击「新增配方」，填写名称并选择类型
3. 进入配方编辑视图
4. 在底部快捷行添加材料：选择材料 → 输入用量 → 选择单位
5. （可选）设置损耗率（如 5% 的料头损耗）
6. 保存后系统自动计算成本

### 配方状态

| 状态 | 含义 |
|------|------|
| 待完善 | 配方尚无明细 |
| 待核对 | 有明细但材料缺少批次成本 |
| 可出品 | 成本完整，可以使用 |
| 已绑定菜单 | 已关联到菜品，正式启用 |

### 注意事项

- 修改被其他配方引用的半成品时，系统会弹出警告
- 删除配方前系统会检查是否有其他配方引用

## 库存管理

### 批次入库

1. 进入「库存」页面
2. 点击「新增批次」
3. 填写：材料、批次号（可自动生成）、数量、单价
4. （可选）填写生产日期、保质期、供应商

### 库存扣减方式

系统支持两种扣减触发方式：

| 方式 | 触发时机 |
|------|----------|
| 订单扣减 | KDS 完成出餐时，按配方自动扣减 |
| 生产扣减 | 生产单完成时，按配方扣减原料 |

### 库存调整

- **手工调整**：直接修改某批次数量（如发现少了）
- **废弃记录**：记录损耗，同时扣减库存
- **盘点**：创建盘点单 → 录入实际数量 → 系统自动计算差异 → 确认后调整

### 预警

- 材料总库存低于设定阈值时，仪表板自动提醒
- 批次即将到期时显示临期预警

## 采购与生产

### 采购流程

```
创建采购单 → 添加材料明细 → 发送给供应商 → 收货入库 → 自动生成批次
```

1. 进入「采购单」页面，点击「新增采购单」
2. 选择供应商，添加需要采购的材料及数量
3. 收到货物后，点击「确认收货」
4. 系统自动创建库存批次并记录入库流水

### 生产流程

```
创建生产单 → 选择半成品配方 → 填写计划数量 → 开始生产 → 完成并入库
```

1. 进入「生产单」页面，点击「新增生产单」
2. 选择要生产的半成品配方
3. 填写计划生产数量
4. 点击「开始」→ 完成后点击「完工」
5. 系统自动扣减原料库存，并将成品入库

## 报表分析

进入「报表」页面，可查看以下维度的分析数据：

| 报表 | 内容 |
|------|------|
| 销售趋势 | 每日/每周/每月的销售额变化 |
| 毛利分析 | 收入 vs 食材成本，毛利率 |
| 热销排行 | 各菜品销售数量排名 |
| 分类统计 | 各分类的销售占比 |
| 原料消耗 | 各材料的使用量统计 |

> 💡 仪表板首页也提供今日/本周/本月的快速概览。

## 打印设置

### 支持的打印方式

| 方式 | 说明 |
|------|------|
| LAN 局域网打印 | 通过 IP 直连热敏打印机 |
| 飞鹅云打印 | 通过云端 API 远程打印 |

### 配置打印机

1. 进入「打印设置」页面
2. 点击「新增打印机」
3. LAN 打印：输入打印机 IP 和端口（可使用「扫描局域网」自动发现）
4. 飞鹅打印：输入飞鹅账号信息（User、UKey、SN、Key）
5. 选择纸宽（58mm / 80mm）
6. 可设为默认打印机

### 打印模板

- 在「打印模板」页面可自定义各类单据的模板
- 支持预览功能

## 数据备份

### 手动备份

所有数据保存在单一 SQLite 文件中，直接复制即可备份：

| 平台 | 文件位置 |
|------|----------|
| macOS | `~/Library/Application Support/Cuckoo/cuckoo.db` |
| Windows | `%LOCALAPPDATA%\Cuckoo\cuckoo.db` |

**备份方法**：
1. 关闭 Cuckoo 应用
2. 复制 `cuckoo.db` 文件到安全位置（U盘、网盘等）
3. 建议每天营业结束后备份一次

**恢复方法**：
1. 关闭 Cuckoo 应用
2. 将备份的 `cuckoo.db` 覆盖回原位置
3. 重新打开 Cuckoo

## 常见问题

### macOS 打开提示"无法验证开发者"

前往 **系统设置 → 隐私与安全性**，在最下方找到被阻止的应用，点击「仍要打开」。

### 打印机连接不上

1. 确认打印机和电脑在同一局域网
2. 尝试使用「扫描局域网」功能查找打印机
3. 检查打印机 IP 是否正确（可在打印机上打印自检页查看）
4. 检查端口是否为 9100（热敏打印机默认端口）

### 库存数据不对

1. 检查配方中材料的用量和单位是否正确
2. 检查是否有未完成的订单占用了预扣库存
3. 使用「盘点」功能进行实物盘点并修正差异

### 配方成本显示为 0

- 确保配方中使用的材料已有入库批次（批次中包含单价信息）
- 配方状态为「待核对」时表示有材料缺少成本数据
- 点击配方中的提示可直接跳转到库存页补充批次成本

### 如何获取帮助

如果遇到无法解决的问题：
1. 进入「设置」页面
2. 点击「复制诊断报告」
3. 将报告内容发送给技术支持

> 详细的日志获取方法见 [远程协助指南](remote-assistance-guide.md)
